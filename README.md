# Choice
## サイト概要
慣れてしまった日常生活の中に新しい選択肢を生むウェブサイト。自分の日常をイベント化して参加者を募集できるアプリケーションです。<br>
変化のない日々を過ごすことに飽き飽きしたことはありませんか？オリジナルの使い方でありふれた日常に変化を与えましょう。<br>

## 制作背景
〜自分の生活に対して変化を求めているが、解決策が見つからない人は多い。〜<br>
- マンネリ化した日常に飽き飽きしている
- 何か始めたいけど難しく考えすぎていい案が浮かばない。
- 勇気が出なくてチャレンジできない。
- 新しい仲間が欲しい。
- 様々な面でスキルアップしたい。等<br>

このような問題を簡単に解決する場所や、自分の持っていない良い考え方（新しい発見）を得られるような場所が<br>
作れないかなと思い制作することにしました。

## 使用技術
### 開発環境
- OS：macOS Catalina 10.15.7
### 技術
- Ruby 2.6.3 / RubyOnRails 5.2.4
- HTML&CSS(Sass,Bootstrap)
- javascript,jQuery
- AWS（VPC, EC2, RDS, Route53, ALB, ACM）
- Nginx, Puma
- MySQL
- GithubActions（開発環境変更時のデプロイ自動化用アクション）

## 機能一覧
### ユーザー側
 - ユーザー認証機能（Devise)(新規登録,ログイン,ログアウト,退会（論理削除））
 - myページ,ユーザー情報編集
 - イベント関係機能（新規投稿、編集、削除、一覧、詳細表示）
 - 画像アップロード機能（refile)
 - 投稿画像解析、タグ自動生成機能(Google Vision API)
 - 類似投稿提案機能（↑で生成したタグを利用した機能）
 - 画像プレビュー機能（非同期通信）
 - コメント＆いいね機能（コメント＆いいね　投稿、削除 各投稿にいいねコメント数表示）※非同期通信
 - DM機能（トーク相手の一覧表示,トークルーム詳細表示）
 - 検索機能（複数の検索条件を指定可能）
 - 画像スライド機能（slick）
 - 通報機能(違反行動をしているユーザーを管理者に通報できる）
 - ブログ閲覧機能(管理者側で投稿されたブログを閲覧できる）
 - ページネーション機能(kaminari)
### 管理者側
 - 管理者認証機能（Devise)(ログイン,ログアウト）
 - myページ,ユーザー情報編集
 - ブログ関係機能（新規投稿、編集、削除、一覧、詳細表示）
 - ユーザー管理機能（登録ユーザー情報確認、強制退会機能）
 - ページネーション機能(kaminari)
